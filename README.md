# 🔐 Random Password Generator

https://pixelmichal.github.io/Password-generator/

A simple web application that generates secure random passwords based on user preferences. Built with HTML, CSS, JavaScript and Bootstrap.

---

## 📸 Preview

The app allows users to:

- choose password length (4–24)
- include uppercase letters
- include numbers
- include special characters
- generate and copy password to clipboard

---

## 🚀 Features

- Generate random passwords instantly
- Select password length (4–24 characters)
- Choose character types:
  - ABC (uppercase letters)
  - 123 (numbers)
  - !@# (special characters)
- Copy password to clipboard
- Visual "COPIED" alert confirmation
- Responsive layout using Bootstrap

---

## 🧠 How it works

The generator builds a password by:

1. Creating a base character pool
2. Randomly selecting characters
3. Ensuring selected checkbox rules are applied
4. Returning a final password string
